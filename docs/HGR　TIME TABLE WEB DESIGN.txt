	    그리고 이름, 학번, 수업반 같은 필수 입력 사항 기재 목록 제시

	2. 필수 입력 사항 기재 목록 옆에서 오버 플로우 로고 표시
	    로딩 홈페이지 ? 오버플로우 빙글빙글

	3. 이름 입력 ? 프린트 홈페이지에 이름 기재 (펀의 사항)
	    학번 입력 ? 프린트 홈페이지에 학번 기재 (편의 사항)
			학번을 array로 전환 후 (0,1,2,3,4)
			0은 학년을 알아보는 기준으로 삼음
	    수업반 입력 ? 1차 classifier

	4. 계산 중 막히는 구간 -수업반이 겹치는 구간-에서 추가 질문
	    ㄱ. 당신의 수업반은 ... 이다(선택)
 		→ 듣는 과목은 집합 Choices else 집합 notChices
	    ㄴ. 이후 같은 질문에서 동일한 선택지로 자동 설정
	    ㄷ. 1배열의 경우 바로 적용
	    ㄹ. 0배열의 경우 공강
	    ㅁ. 6교시 0배열은 해당 요일 야자 여부를 물어보고 귀가 or 야자
	    ㅂ. 모든 배열의 과목이 집합 B인 경우 공강
	    ㅅ. 모든 배열의 과목이 집합 A인 경우 해당 수업(오류 방지)
